Как запретить доступ по ssh для рута по паролю

23-01-2021

Хорошо, что у нас на сервере есть панель ISPManager.

Это даст нам возможность запретить ssh доступ для пользователя root по паролю.

И разрешить доступ только по ключам (ssh-keys).

Итак, входим в панель ISPManager рутом. Находим Filemanager и находим папку 

/etc/ssh/

Там находим файл sshd_config и открываем для редактирования.

Там находим строчку PasswordAuthentication и ставим значение no

Сохраняем файл и теперь надо перезапустить ssh на сервере.

Это можно сделать командой

systemctl restart ssh

Теперь по паролю рут не зайдет на сервер.

Но, нам надо подключить авторизацию по ключам.

А перед этим надо Сгеренировать ssh key
