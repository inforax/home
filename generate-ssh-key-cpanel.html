Как создать ssh key в панели CPanel

13-06-2022

Если выполните шаги инструкции правильно, то сможете создать и скачать файл ssh key, который (если вы на Windows) нужно будет положить в папку C:/Users/Ваш пользователь/.ssh

SSH Key - это файл, с с помощью которого вы будете авторизоваться на сервер по ssh. Это наиболее безопасный способ для входа на ваш сервер.

Чтобы создать ssh key в CPanel для доступа по ssh, нужно проделать следующие шаги:

  1. Авторизуемся в CPanel
  2. Ищем раздел Security, а там тыкаем в SSH Access

3. Там тыкаем в Manage SSH Keys

4. Дальше жмем Generate a New Key

5. Вводим название файла, пароль и все остальное как на скрине.

ВНИМАНИЕ! Не забудьте сохранить пароль, он будет нужен вам всегда, когда вы задумаете войти на сервер по SSH

6. Теперь возвращаемся назад. У вас должно так: (ну или почти так)

7. Теперь надо авторизовать ключ, жмем на кнопку Manage

8. Там жмете кнопку Authorize (у меня ssh key уже авторизован)

9. Теперь осталось скачать файл и всё.
